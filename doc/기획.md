플랫폼 기획안: "무빙애니멀 마켓" (가칭)
1. 핵심 기능 구조
📱 랜딩 페이지 (/)
├── 🐰 움직이는 동물 캐릭터 히어로 섹션
├── 📊 플랫폼 소개
└── 🎯 인기 캐릭터 미리보기

🛍️ 마켓플레이스 (/marketplace)
├── 카테고리별 필터 (토끼, 고양이, 강아지 등)
├── 가격대별 정렬
└── 애니메이션 미리보기 카드

📦 상품 상세 (/product/[id])
├── 실시간 애니메이션 데모
├── 캐릭터 정보 (크기, 포맷, 애니메이션 종류)
├── 구매/장바구니 버튼
└── 유사 상품 추천

🛒 장바구니 & 결제 (/cart, /checkout)

👤 사용자 페이지 (/my-characters)
└── 구매한 캐릭터 다운로드
2. 기술 스택 제안
애니메이션 구현:

Framer Motion (React 애니메이션)
Lottie (JSON 기반 애니메이션)
CSS Animations + Tailwind
상태 관리:

React Context (장바구니)
localStorage (임시 저장)
결제:

PortOne (구 아임포트) - 한국 결제 지원
Stripe (글로벌)
데이터:

초기: JSON 파일
추후: Supabase / Firebase
3. 캐릭터 데이터 구조 예시
interface AnimalCharacter {
  id: string
  name: string
  category: 'rabbit' | 'cat' | 'dog' | 'bird'
  animationType: 'bounce' | 'wave' | 'spin' | 'walk'
  price: number
  previewUrl: string  // GIF/Video
  fileUrl: string     // 실제 애니메이션 파일
  format: 'lottie' | 'gif' | 'svg' | 'video'
  tags: string[]
}
4. 단계별 개발 계획
Phase 1 (MVP - 1-2주):

✅ 랜딩 페이지 + 움직이는 토끼 캐릭터 (Framer Motion)
✅ 간단한 상품 목록 페이지 (5-10개 샘플)
✅ 상품 상세 페이지
✅ 기본 UI/UX 디자인
Phase 2 (결제 - 1주):

장바구니 기능
결제 시스템 연동
이메일 전송 (구매 확인)
Phase 3 (고도화 - 2주):

사용자 계정 시스템
검색 & 필터링
리뷰 시스템
관리자 대시보드
5. 디자인 컨셉
🎨 컬러: 파스텔 톤 + 생동감 있는 포인트 컬러
🎭 분위기: 귀엽고 친근한, 하지만 전문적인
📱 반응형: 모바일 퍼스트
✨ 인터랙션: 호버 시 캐릭터가 움직임
